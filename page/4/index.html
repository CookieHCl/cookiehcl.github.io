Vector space

양자역학에선 복소수를 사용함

vector addition ∣V⟩+∣W⟩\ket{V} + \ket{W}∣V⟩+∣W⟩와 multiplication by scalar a∣V⟩a\ket{V}a∣V⟩ 연산이 있고, 아래 조건들을 만족하는 공간을 vector space라고 함

  • Closure: vector addition과 multiplication by scalar의 결과도 vector space에 속해야 함
  • Scala multiplication is distributive: a(∣V⟩+∣W⟩)=a∣V⟩+a∣W⟩,(a+b)∣V⟩=a∣V⟩+b∣V⟩a(\ket{V}+\ket{W}) = a\ket{V} + a\ket{W}, (a+b)\ket{V} = a\ket{V} + b\ket{V}a(∣V⟩+∣W⟩)=a∣V⟩+a∣W⟩,(a+b)∣V⟩=a∣V⟩+b∣V⟩
  • Scala multiplication is associative: a(b∣V⟩)=(ab)∣V⟩a(b\ket{V}) = (ab)\ket{V}a(b∣V⟩)=(ab)∣V⟩
  • Addition is commutative: ∣V⟩+∣W⟩=∣W⟩+∣V⟩\ket{V} + \ket{W} = \ket{W} + \ket{V}∣V⟩+∣W⟩=∣W⟩+∣V⟩
  • Addition is associative: ∣V⟩+(∣W⟩+∣Z⟩)=(∣V⟩+∣W⟩)+∣Z⟩\ket{V} + (\ket{W} + \ket{Z}) = (\ket{V} + \ket{W}) + \ket{Z}∣V⟩+(∣W⟩+∣Z⟩)=(∣V⟩+∣W⟩)+∣Z⟩
  • Existence of additive identity: ∣0⟩∈V\ket{0} \in V∣0⟩∈V such that ∣V⟩+∣0⟩=∣V⟩\ket{V} + \ket{0} = \ket{V}∣V⟩+∣0⟩=∣V⟩
  • Existence of additive inverse: ∣−V⟩∈V\ket{-V} \in V∣−V⟩∈V such that ∣V⟩+∣−V⟩=∣0⟩\ket{V} + \ket{-V} = \ket{0}∣V⟩+∣−V⟩=∣0⟩
  • Existence of scalar multiplication identity: 1∣V⟩=∣V⟩1\ket{V} = \ket{V}1∣V⟩=∣V⟩
