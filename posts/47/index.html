Discrete Time Fourier Transform

Discrete time에서 함수를 N차원 벡터 ∣x⟩=[x0,x1,…,xN−1]\ket{x} = [x_0, x_1, \ldots, x_{N-1}]∣x⟩=[x0​,x1​,…,xN−1​]으로 나타낼 수 있음

이때 orthonormal basis ∀0≤k<N,∣k⟩=1N[e−2πik⋅0N,e−2πik⋅1N,…,e−2πik⋅(N−1)N]\forall 0 \leq k < N, \ket{k} = \frac{1}{\sqrt{N}} \left[ e^{-2\pi i \frac{k \cdot 0}{N}}, e^{-2\pi i \frac{k \cdot 1}{N}}, \ldots, e^{-2\pi i \frac{k \cdot (N-1)}{N}} \right]∀0≤k<N,∣k⟩=N​1​[e−2πiNk⋅0​,e−2πiNk⋅1​,…,e−2πiNk⋅(N−1)​]를 생각하자. (참고: e−2πikt=cos⁡(2πkt)−isin⁡(2πkt)e^{-2\pi ikt} = \cos(2\pi kt) - i\sin(2\pi kt)e−2πikt=cos(2πkt)−isin(2πkt), continuous fourier transform에서 discrete니까 sampling을 x=TjN,0≤j<Nx = \frac{Tj}{N}, 0 \leq j < Nx=NTj​,0≤j<N에서 했다고 생각하면 됨)

orthonormal basis이므로 ∣x⟩=∑ck∣k⟩\ket{x} = \sum c_k\ket{k}∣x⟩=∑ck​∣k⟩처럼 계수를 구할 수 있음 (Recall: ck=⟨k∣x⟩c_k = \braket{k \vert x}ck​=⟨k∣x⟩)

∣x⟩=x0∣0⟩+x1∣1⟩+⋯+xN−1∣N−1⟩\ket{x} = x_0\ket{0} + x_1\ket{1} + \cdots + x_{N-1}\ket{N-1}∣x⟩=x0​∣0⟩+x1​∣1⟩+⋯+xN−1​∣N−1⟩을 ∣y⟩=y0∣0⟩+y1∣1⟩+⋯+yN−1∣N−1⟩\ket{y} = y_0\ket{0} + y_1\ket{1} + \cdots + y_{N-1}\ket{N-1}∣y⟩=y0​∣0⟩+y1​∣1⟩+⋯+yN−1​∣N−1⟩로 변환하는 것을 QFT라고 함
