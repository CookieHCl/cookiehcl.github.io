Scrambling Techniques

DC component를 없애고 synchronization, error detection을 위해 일부러 신호를 다르게 보냄

B8ZS(Bipolar with 8-Zeros Substitution)

bipolar-AMI인데 0 8개는 000+-0-+나 000-+0+-로 보냄

--나 ++가 겹치기 때문에 실제로는 violation(불가능한 신호); 이런 신호가 들어오면 0 8개로 해석해야 함

HDB3(High Density Bipolar 3-Zeros)

bipolar-AMI인데 0 4개를 000-, 000+, +00+, -00- 중 하나로 보냄

4번째 비트가 반드시 앞 신호와 부호가 바뀐 violation이 되도록 신호를 보냄

  • 0000 직전 신호가 -인데 violation 이후 1이 홀수개면 (즉, 마지막 violation이 +면) 000-
  • 0000 직전 신호가 -인데 violation 이후 1이 짝수개면 (즉, 마지막 violation이 -면) +00+
  • 0000 직전 신호가 +인데 violation 이후 1이 홀수개면 (즉, 마지막 violation이 -면) 000+
  • 0000 직전 신호가 +인데 violation 이후 1이 짝수개면 (즉, 마지막 violation이 +면) -00-

Digital Data -> Analog Signal

진폭 A, 진동수 f, 위상 θ만 정해지면 신호 Acos⁡(2πft+θ)A\cos(2\pi ft + \theta)Acos(2πft+θ)가 특정됨

Modulation rate(baud)
신호를 보내는 속도

Data rate = Modulation rate * 신호당 비트 수

ASK

0이면 Acos⁡(2πft+θ)A\cos(2\pi ft + \theta)Acos(2πft+θ), 1이면 0

FSK

0이면 Acos⁡(2πft+θ)A\cos(2\pi ft + \theta)Acos(2πft+θ), 1이면 Acos⁡(2πf′t+θ)A\cos(2\pi f't + \theta)Acos(2πf′t+θ)

PSK

0이면 Acos⁡(2πft)A\cos(2\pi ft)Acos(2πft), 1이면 Acos⁡(2πft+180°)A\cos(2\pi ft + 180\degree)Acos(2πft+180°)

QPSK (Quadrature PSK)

11이면 2cos⁡(2πft+π4)\sqrt{2}\cos(2\pi ft + \frac{\pi}{4})2​cos(2πft+4π​),
01이면 2cos⁡(2πft+3π4)\sqrt{2}\cos(2\pi ft + \frac{3\pi}{4})2​cos(2πft+43π​),
00이면 2cos⁡(2πft+−3π4)\sqrt{2}\cos(2\pi ft + \frac{-3\pi}{4})2​cos(2πft+4−3π​)
10이면 2cos⁡(2πft+−π4)\sqrt{2}\cos(2\pi ft + \frac{-\pi}{4})2​cos(2πft+4−π​)

실제 구현) 앞 비트를 I, 뒷 비트를 Q라고 했을 때 0이면 -1, 1이면 1로 매핑한 다음 Icos⁡(2πft)−Qsin⁡(2πft)I\cos(2\pi ft) - Q\sin(2\pi ft)Icos(2πft)−Qsin(2πft)로 보냄

받을 때는 주어진 신호에 각각 cos⁡(2πft)\cos(2\pi ft)cos(2πft)랑 −sin⁡(2πft)-\sin(2\pi ft)−sin(2πft)를 곱하고 Low pass filter를 통과시키면(즉, ∫0T\int_0^T∫0T​ 적분을 하면) T2\frac{T}{2}2T​가 나왔을 때 I/Q는 1, −T2-\frac{T}{2}−2T​가 나왔을 때 I/Q는 0임

Offset-QPSK modulation

원래 위 과정대로 받으면 IQ가 동시에 나옴;

Q 부분만 (즉, −sin⁡(2πft)-\sin(2\pi ft)−sin(2πft) 곱한 부분만) delay를 줘서 clock 2배로 IQ신호가 번갈아서 나오게 함
