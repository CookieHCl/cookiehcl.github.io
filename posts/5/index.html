양자역학 공리 1

The state of the particle is represented by a vector ∣ψ(t)⟩\ket{\psi(t)}∣ψ(t)⟩ in a Hilbert space.

Hilbert space는 inner product space + metric 존재

사실 어떤 state space인지도 모름 -> 실험으로 정함 (ex: ∣0⟩,∣1⟩\ket{0}, \ket{1}∣0⟩,∣1⟩)

Two-level atom(TLA)

원자 에너지 준위가 2개인 모델; 높으면 ∣1⟩\ket{1}∣1⟩, 낮으면 ∣0⟩\ket{0}∣0⟩

전자의 일반적인 에너지는 ∣ψ⟩=α∣0⟩+β∣1⟩\ket{\psi} = \alpha\ket{0} + \beta\ket{1}∣ψ⟩=α∣0⟩+β∣1⟩

빛의 편광

수평 전자기파랑 수직 전자기파랑 위상차이로 인해 상태가 달라짐

  • ∣H⟩\ket{H}∣H⟩ horizontal
  • ∣V⟩\ket{V}∣V⟩ vertical
  • ∣D⟩=∣H⟩+∣V⟩2\ket{D} = \frac{\ket{H} + \ket{V}}{\sqrt{2}}∣D⟩=2​∣H⟩+∣V⟩​ diagonal
  • ∣A⟩=∣H⟩−∣V⟩2\ket{A} = \frac{\ket{H} - \ket{V}}{\sqrt{2}}∣A⟩=2​∣H⟩−∣V⟩​ anti-diagonal
  • ∣R⟩=∣H⟩+i∣V⟩2\ket{R} = \frac{\ket{H} + i\ket{V}}{\sqrt{2}}∣R⟩=2​∣H⟩+i∣V⟩​ right-circular (waveform이 시계방향으로 돌아감)
  • ∣L⟩=∣H⟩−i∣V⟩2\ket{L} = \frac{\ket{H} - i\ket{V}}{\sqrt{2}}∣L⟩=2​∣H⟩−i∣V⟩​ left-circular
