CurryвҖ“Howard correspondence

Functional Programming LanguageлҠ” Mathematicsлһ‘ мқјлҢҖмқј лҢҖмқ‘лҗңлӢӨ!

P: Tлқјл©ҙ T(Type)мқҖ proposal(лӘ…м ң), P(Program)мқҖ proof(мҰқлӘ…)мқҙлқјкі  мғқк°Ғн•  мҲҳ мһҲлӢӨ.
л§Ңм•Ҫ type checkingмқҙ лҗңлӢӨл©ҙ, proposalм—җ л§һлҠ” proofк°Җ мһҲлӢӨлҠ” лң»мқҙлҜҖлЎң, лӘ…м ңк°Җ м°ёмқҙлӢӨ!

мқҙкұё мҷң н•ҳлғҗ?
м—„л°Җн•ҳкІҢ мҲҳн•ҷ, лҳҗлҠ” н”„лЎңк·ёлһЁмқҳ м •нҷ•м„ұмқ„ мҰқлӘ…н•  мҲҳ мһҲлӢӨ!
нҠ№нһҲ LLM мӢңлҢҖм—җ LLMмқҙ лӮҙлҶ“мқҖ м¶ңл Ҙмқ„ мӮ¬мӢӨ лҜҝмқ„ мҲҳк°Җ м—ҶлҠ” мғҒнҷ©мһ„ -> мҰқлӘ…ліҙмЎ°кё°лЎң кІҖмҰқн•  мҲҳ мһҲлӢӨл©ҙ?

мӢӨм ңлЎң мҲҳн•ҷмһҗл“ӨлҸ„ мҰқлӘ…мқ„ (Coqмқҙ м•„лӢҲлқј Leanмқ„ м“°м§Җл§Ң) м»ҙн“Ён„°лЎң кІҖмҰқн•ҳкі  мһҲлӢӨ!

лҚ” л“Өм–ҙк°Җмһҗл©ҙ T(Type)мқҖ м–ҙл–Ө м§‘н•©мқҙлқјкі  ліј мҲҳ мһҲлҠ”лҚ°, м—¬кё°м—җ мҶҚн•ң element P(Program)мқ„ м°ҫмқ„ мҲҳ мһҲлӢӨл©ҙ TлҠ” м°ёмқҙлқјлҠ” лң»!
л°ҳлҢҖлЎң Tм—җ мҶҚн•ң elementк°Җ м—ҶлӢӨл©ҙ TлҠ” кұ°м§“мқҙлӢӨ!

Rocq Calculus

Rocqм—җлҠ” TypeмқҙлқјлҠ” кІғл§Ң мЎҙмһ¬н•Ё (к·ёлғҘ нӮӨмӣҢл“ңмһ„), лӮҳлЁём§ҖлҠ” м „л¶Җ мӮ¬мҡ©мһҗк°Җ м •мқҳн•ҳлҠ” кІғ!
мӮ¬мӢӨ Proposalмҡ© Propмқҙлһ‘ Valueмҡ© Setмқҙ л”°лЎң мЎҙмһ¬н•ҳкёҙ н•ҳлҠ”лҚ° мқјлӢЁмқҖ л‘ҳ лӢӨ к·ёлғҘ Typeмқҙлқјкі  мғқк°Ғн•ҳкі  л¬ҙмӢң
мӢӨм ңлЎң Typeмқҙлһ‘ лҳ‘к°ҷмқҖлҚ° к·ёлғҘ нҺёмқҳмғҒ кө¬л¶„н•ҙлҶ“мқҖ кІғ

нғңмҙҲм—җлҠ” TypeмқҙлқјлҠ” кІғл§Ң мЎҙмһ¬ (к·ёлғҘ нӮӨмӣҢл“ңмһ„)
лӮҳлЁём§ҖлҠ” м „л¶Җ мӮ¬мҡ©мһҗк°Җ м •мқҳн•ҳлҠ”кұ°

TypeмқҖ лӘЁл“  setмқҳ set,
MyType: TypeмқҖ нҠ№м • set,
MyElement: MyTypeлҠ” к·ё setмқҳ element

лӘЁл“  elementлҠ” л”ұ н•ҳлӮҳмқҳ Typeм—җл§Ң мҶҢмҶҚлҗҳм–ҙмһҲмқҢ!!!
Rocqм—җ subtypeлҠ” мЎҙмһ¬н•ҳм§Җ м•ҠлҠ”лӢӨ!
(мҳҲмҷёлЎң Prop, SetмқҖ Typeмқҳ subtypeмқҙм§Җл§Ң к·ёлғҘ Typeмқҙлқјкі  мғқк°Ғн•ҙлҸ„ лҗЁ)
мҰү, 1мқҙ intмқҙл©ҙм„ң shortмқҙл©ҙм„ң boolмқјмҲҳлҠ” м—ҶлӢӨлҠ”кұ°мһ„

Inductive

InductiveлЎң setкіј к·ё setмқҳ elementлҘј м •мқҳн•  мҲҳ мһҲмқҢ.
CheckлҘј нҶөн•ҙ Type нҷ•мқё к°ҖлҠҘ!

м—¬кё°м„ң mondayлҠ” setмқҳ elementлҘј м •мқҳн•ң кІғмқҙкё° л•Ңл¬ём—җ Check (monday: Type).лҠ” мӢӨнҢЁн•ҳкІҢ лҗЁ

Definition

DefinitionмңјлЎң н•ЁмҲҳлҘј м •мқҳн•  мҲҳ мһҲмқҢ.
н”„лЎңк·ёлһЁмқ„ кі„мӮ°н–Ҳмқ„ л•Ң к°ҷлӢӨ = мҲҳн•ҷм—җм„ң лҸҷм№ҳкҙҖкі„лҘј мқҳлҜён•Ё.
e.g. next_working_day fridayлҠ” mondayлһ‘ лҸҷм№ҳ!

н•ҳм§Җл§Ң н”„лЎңк·ёлһЁмқҙлӢӨ ліҙлӢҲк№Ң "кі„мӮ°"мқҖ м•„л¬ҙ лҸҷм№ҳкҙҖкі„лЎңлӮҳ мӣҖм§Ғмқј мҲҳ мһҲлҠ”кІҢ м•„лӢҲлқј м ңмқј лӢЁмҲңн•ҙм§ҖлҠ” л°©н–ҘмңјлЎң (мҰү, reduced formмңјлЎң) кі„мӮ°лҗЁ.
мҰү mondayк°Җ к°‘мһҗкё° next_working_day fridayлЎң кі„мӮ°лҗ  мҲҳлҠ” м—ҶмқҢ!

л§Ңм•Ҫ T1: Typeл©ҙм„ң T2: Typeмқҙлқјл©ҙ T1->T2: Typeмһ„
мҰү T1->T2лҸ„ setмһ„! T1 setм—җм„ң T2 setмңјлЎң к°ҖлҠ” н•ЁмҲҳл“Өмқҳ м§‘н•©мқ„ лӮҳнғҖлғ„.
e.g. next_working_day: day -> day

InductiveмІҳлҹј T1->T2 setмқҳ мӣҗмҶҢлҘј л°”лЎң л§Ңл“Өм–ҙлӮҙкі  мӢ¶лӢӨл©ҙ lambda functionмқ„ мӮ¬мҡ©н•  мҲҳ мһҲмқҢ.

мӮ¬мӢӨ DefinitionмқҖ м—„л°ҖнһҲ л”°м§Җл©ҙ н•ЁмҲҳлҘј м •мқҳн•ҳлҠ” syntaxк°Җ м•„лӢҲлқј lambda functionм—җ мқҙлҰ„ л¶ҷмқҙлҠ” syntax sugarмһ„...

мӮ¬мӢӨ мқҙкұҙ

мқҙкұ°лһ‘ лҳ‘к°ҷмқҢ!

м—¬лӢҙ) syntaxмқҳ м§‘н•©мқҖ countableмқҙм§Җл§Ң, setм—җм„ң setмңјлЎң к°ҖлҠ” н•ЁмҲҳмқҳ м§‘н•©мқҖ uncountableмһ„.
мҰү, мҲҳн•ҷм ҒмңјлЎң мЎҙмһ¬н•ҳм§Җл§Ң мҡ°лҰ¬к°Җ RocqлЎң н‘ңнҳ„н•  мҲҳ м—ҶлҠ” н•ЁмҲҳк°Җ мЎҙмһ¬н•Ё! (к·ёкІғлҸ„ uncountable л§ҢнҒјмқҙлӮҳ мЎҙмһ¬н•Ё!!)
н•ҳм§Җл§Ң мғқк°Ғн•ҙліҙл©ҙ көімқҙ Rocqк°Җ м•„лӢҲлқј м–ҙл–Ө м–ём–ҙм—¬лҸ„ (мӢ¬м§Җм–ҙ мһҗм—°м–ҙм—¬лҸ„) м„ұлҰҪн•ҳлҠ” л§җмқҙлӢӨ..?

match

мҷ м§Җ лӘЁлҘҙкІ лҠ”лҚ° мҲҳн•ҷм—җлҠ” dualмқҙ мһҲм–ҙм•јн•Ё (мӮ¬мӢӨ мҲҳн•ҷмһҗлҠ” лӘ°лһҗлҠ”лҚ° м»ҙкіөн•ңн…Ңм„ң мһҗм—°мҠӨлҹҪкІҢ л°°мӣҖ)
Setмқҳ elementлҘј л§Ңл“Ө мҲҳлҸ„ мһҲлӢӨл©ҙ elementлҘј м—Ҷм•Ё мҲҳлҸ„ мһҲм–ҙм•ј н•Ё?

н”„лЎңк·ёлһЁмқ„ "кі„мӮ°"н•ҳл©ҙ elementлҘј м—Ҷм• лІ„лҰј.
к·ё мӨ‘ InductiveлҠ” matchлЎң м ңкұ°н•Ё.

  • value: кі„мӮ°мқҙ мҷ„м „нһҲ лҒқлӮң к°’
  • expression: кі„мӮ°мқ„ н•ҳл©ҙ к°’мқҙ лҗЁ

Inductiveмқҳ valueлҠ” мІҳмқҢм—җ м •мқҳн–ҲлҚҳ к°’л“Өл°–м—җ м—ҶмқҢ!!
e.g. dayмқҳ valueлҠ” monday ~ sundayл°–м—җ м—ҶмқҢ.

apply

Functionмқҳ elimination ruleмқҖ applyмһ„...
л§җ к·ёлҢҖлЎң f xн•ҳл©ҙ functionмқ„ мӮ¬мҡ©н•ң кІғ!

Compute

мҡ”м•Ҫ) Typeм—җлҠ” Inductiveлһ‘ Functionмқҙ мһҲмқҢ
к°Ғ Typeл§ҲлӢӨ introduction ruleмқҙлһ‘ elimination ruleмқҙ мһҲмқҢ
мқҙ л„Ө к°Җм§Җ ruleмқ„ type checkingмқҙ к°ҖлҠҘн•ҳлҸ„лЎқ мЎ°н•©н•ң лӘЁл“  кІғмқ„ expressionмқҙлқјкі  л¶ҖлҰ„

Inductiveмқҳ introductionмқҖ Inductive
Inductiveмқҳ eliminationмқҖ match
Functionмқҳ introductionмқҖ lambda function
Functionмқҳ eliminationмқҖ apply (нӮӨмӣҢл“ңлҠ” м•„лӢҲм§Җл§Ң...)

мқҙм ң кі„мӮ°мқ„ мўҖ лҚ” м—„л°Җн•ҳкІҢ м •мқҳн•  мҲҳ мһҲлҠ”лҚ° к°ҷмқҖ type(Inductive/Function)м—җ лҢҖн•ҙм„ң introductionкіј eliminationмқҙ л§ҢлӮ  кІҪмҡ° reductionмқҙ мқјм–ҙлӮЁ.

e.g. м•„лһҳ мҪ”л“ңлҘј мӢӨн–үмӢңнӮӨл©ҙ

лЁјм Җ н•ЁмҲҳ argumentлҘј лҢҖмІҙн•ҳкі 

matchлҘј л§һм¶”кІҢ лҗЁ.

м—¬лӢҙ) Proof

мқҙкІғл“Өмқ„ мқҙмҡ©н•ҙм„ң мҰқлӘ…мқ„ м–ҙл–»кІҢ н•ҳлғҗ?

eq_reflмқҖ м–‘ліҖмқҙ к°ҷмқҖ л“ұмӢқл“Өмқҳ м§‘н•©мһ„.
л§Ңм•Ҫ (next_working_day friday)к°Җ mondayк°Җ м•„лӢҲлқјл©ҙ eq_reflм—җ мҶҚн•ҳм§Җ м•ҠмңјлҜҖлЎң type checkingм—җ мӢӨнҢЁн•Ё!
л°ҳлҢҖлЎң type checkingмқҙ лҗңлӢӨл©ҙ (next_working_day friday)к°Җ mondayмҷҖ к°ҷлӢӨлҠ”кІҢ мҰқлӘ…мқҙ лҗҳлҠ” кІғ

мӢӨм ң мҰқлӘ…мқҖ мқјмқјнһҲ typeмқ„ м§Җм •н•ҙмЈјлҠ”кІҢ м•„лӢҲлқј мқҙлҹ° мӢқмңјлЎң мҰқлӘ…н•ҳм§Җл§Ң... мқҙлҠ” нӣ„мҲ н•Ё

Standard library

RocqлҠ” мӮ¬мӢӨ мң„м—җ мһҲлҠ” л„Ө к°Җм§Җ ruleкіј кі„мӮ°л§ҢмңјлЎң м •мқҳлҗҳм–ҙмһҲмқҢ!
н•ҳм§Җл§Ң мҶҗк°ҖлқҪмқҳ мҲҳлӘ…мқ„ мң„н•ҙм„ң м•„мЈј лӢӨм–‘н•ң syntax sugarл“Өкіј, (e.g. DefinitionмңјлЎң н•ЁмҲҳ introduction)
лҜёлҰ¬ м •мқҳн•ҙл‘” лӢӨм–‘н•ң standard libraryк°Җ мһҲмқҢ.

м—„л°ҖнһҲ л”°м§Җл©ҙ м—¬кё° мһҲлҠ” лӘЁл“  лӮҙмҡ©мқҙ standard libraryмқёкұҙ м•„лӢҲкёҙ н•Ё;;

bool

Rocqм—җлҠ” boolкіј лӢӨм–‘н•ң н•ЁмҲҳл“Өмқҙ м •мқҳлҗҳм–ҙмһҲмқҢ.
мң„ мҪ”л“ңлҠ” мӢӨм ң кө¬нҳ„мқ„ л”°лқјн•ң кІғ!

мһ л§Ң andb, orbлҠ” лӯҗ м–ҙл–»кІҢ мғқкІЁлЁ№мқҖкұ°мһ„?

мӮ¬мӢӨ мқҙкұ°лһ‘ к°ҷмқҖкұ°мһ„ boolмқ„ л°ӣм•„м„ң bool -> bool funмқ„ лҰ¬н„ҙ
мөңмў… нғҖмһ…мқҖ bool -> bool -> bool

м°ёкі лЎң л¬ҙн•ң лЈЁн”„лЎң л№ м§ҖлҠ” н•ЁмҲҳлҠ” outputмқҙ мҲҳн•ҷм ҒмңјлЎң кІ°м •мқҙ лҗҳм§Җ м•ҠмқҢ..
к·ёлһҳм„ң RocqлҠ” л¬ҙн•ң лЈЁн”„лЎң л№ м§Ҳ к°ҖлҠҘм„ұмқҙ мһҲлҠ” н•ЁмҲҳлҠ” м• мҙҲм—җ type checkingмқ„ мӢӨнҢЁн•ҳлҸ„лЎқ л§Ңл“Өм—ҲмқҢ!
(л§Ңм•Ҫ н•ӯмғҒ terminateн•ҳлҠ” н•ЁмҲҳлҘј Rocqк°Җ л°ӣм•„л“Өмқҙм§Җ м•ҠлҠ”лӢӨл©ҙ мҲҳн•ҷм ҒмңјлЎң н•ӯмғҒ terminateн•ңлӢӨлҠ” кІғмқ„ мҰқлӘ…мӢңмјңм•ј type checking к°ҖлҠҘн•Ё;;) мқҙлҹ¬л©ҙ call by value, call by nameмқҳ кІ°кіјк°Җ лҳ‘к°ҷм•„м„ң мӢ кІҪ м•Ҳ м“°кі , лҳҗн•ң reductionмқҳ мҲңм„ңлҘј м–ҙл–»кІҢ н•ҙлҸ„ н•ӯмғҒ к°ҷмқҖ reduced formмңјлЎң кі„мӮ°лҗЁ.

мҰү, bool -> (bool -> bool)лЎң ліҙл“ , (bool -> bool) -> boolлЎң ліҙл“  мғҒкҙҖ м—ҶлӢӨлҠ” мҶҢлҰ¬!
л¬јлЎ  мӢӨм ң type checkerлҠ” лӢЁмҲңн•ң н”„лЎңк·ёлһЁмқҙкё° л•Ңл¬ём—җ reduction мҲңм„ңк°Җ мЎҙмһ¬н•ҳкёҙ н•Ё;

Q) мқҙл ҮкІҢ andлҘј мҚЁлҸ„ лҗҳлҠ”кұ° м•„лӢҳ?

"мӮ¬лһҢмқҙ ліҙкё°м—”" к°ҷмқҖ н•ЁмҲҳк°Җ л§һмқҢ
But кі„мӮ° л°©мӢқмқҙ лӢӨлҘҙкё° л•Ңл¬ём—җ Rocqк°Җ ліҙкё°м—” к°ҷмқҖ н•ЁмҲҳк°Җ м•„лӢҳ...

м •нҷ•нһҲ л§җн•ҳмһҗл©ҙ мқҙ м„ё н•ЁмҲҳк°Җ к°ҷлӢӨлҠ” мӮ¬мӢӨмқҖ мһҗлӘ…н•ҳм§Җ м•ҠмқҢ
Rocqм—җм„ңлҠ” м„ё н•ЁмҲҳк°Җ к°ҷлӢӨлҠ” мӮ¬мӢӨмқ„ мҰқлӘ…н•ҙм•ј н•Ё!

if then else

boolмқҳ matchлҠ” Rocqм—җм„ң if then elseлЎң л°”кҝ”мӨҢ
matchлһ‘ мҷ„м „ лҳ‘к°ҷм§Җл§Ң лӢЁмҲң syntax sugarмһ„

мӮ¬мӢӨ boolмқҖ м—„л°ҖнһҲ л”°м§Җл©ҙ builtinмқҙ м•„лӢҲлқј standard libraryм—җ мҶҚн•Ё!
к·ёлһҳм„ң if then elseлҠ” boolмқҙ м•„лӢҲм—¬лҸ„ м“ё мҲҳ мһҲмқҢ...!! (мӢӨмҡ©м„ұмқҖ кёҖмҺ„..)

м •нҷ•нһҲ clauseк°Җ 2к°ңмқё лӘЁл“  InductiveлҠ” if then else syntaxлҘј мӮ¬мҡ©н•  мҲҳ мһҲмқҢ!
мқҙл•Ң мІ« лІҲм§ё clauseлҘј true(?)лЎң м·Ёкёүн•Ё
